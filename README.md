# Last.fm Scrobble 插件 for Salt Player

此插件为 [Salt Player for Windows](https://moriafly.com/program/spw.html "访问Salt Player for Windows") 添加 Last.fm 音乐记录功能，可自动将您的播放记录提交到 Last.fm。

## 使用说明

### 首次使用设置流程

1. 安装并首次启用插件
2. 插件会自动弹出浏览器窗口，跳转至 Last.fm 授权页面，此时Salt Player 会自动退出
3. 在 Last.fm 页面上授权此应用
4. 授权完成后，重新启动 Salt Player，插件即可正常工作

> ⚠️ 注意：首次启用插件时必须完成 Last.fm 授权才能正常使用功能。授权成功后，后续使用无需再次授权。

### 日常使用

- 插件会在后台自动跟踪播放状态
- 当您播放音乐时，插件会自动提交 Now Playing 信息
- 当歌曲播放进度达到50%时，插件会自动提交 Scrobble 记录
